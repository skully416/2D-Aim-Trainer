JavaScript Data Types: Understanding Primitive and Reference Types

April 25, 2023
Written by Daniel Ogden

Introduction

In JavaScript, data types are used to represent different kinds of values, such as numbers, strings, and objects. Understanding data types is essential for writing effective and error-free JavaScript code.

Primitive Data Types

JavaScript has six primitive data types: 'number', 'string', 'boolean', 'null', 'undefined', and 'symbol'. These data types are called "primitive" because they are immutable, meaning their values cannot be changed once they are created.

Number

The 'number' data type is used to represent numeric values, such as integers and floating-point numbers. For example:

Javascript:

var age = 30;
var price = 4.99;

String

The 'string' data type is used to represent text values, enclosed in quotes (either single or double). For example:

Javascript:

var message = "Hello, world!";

var isActive = false;

Null

Javascript:

var result = null;

Symbol

The 'symbol' data type is used to create unique identifiers for object properties. For example:

Javascript:

var id = Symbol("id");
