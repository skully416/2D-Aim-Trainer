HTML Basics: Part 2

April 25, 2023
Written by Daniel Ogden

Introduction

In the previous tutorial, we covered the basics of HTML and how it is used to create web pages. In this tutorial, we will cover some more advanced topics, including linking to external files, using paragraph and heading elements, and creating lists.

Linking to External Files

HTML allows you to link to external files, such as images, stylesheets, and scripts. To link to an external file, you can use the <link> element in the section of your HTML document, like this:

String

The 'string' data type is used to represent text values, enclosed in quotes (either single or double). For example:

HTML:

<title>My First Web Page</title>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<h1>Welcome to My First Web Page!</h1>
<p>This is the first paragraph of my web page.</p>
<p>This is the second paragraph of my web page.</p>
</body>
</html>

In this example, we are linking to a stylesheet called "styles.css" in the same directory as our HTML file.

Paragraph and Heading Elements

HTML provides several elements for creating paragraphs and headings. Paragraphs are created using the '<p>' element, and headings are created using the '<h1>' through '<h6>' elements.

HTML:

<!DOCTYPE html>
<html>
<head>
<title>My First Web Page</title>
</head>
<body>
<h1>Welcome to My First Web Page!</h1>
<p>This is the first paragraph of my web page.</p>
<h2>Section One</h2>
<p>This is the second paragraph of my web page.</p>
<h3>Subsection One</h3>
<p>This is the third paragraph of my web page.</p>
<h2>Section Two</h2>
<p>This is the fourth paragraph of my web page.</p>
</body>
</html>

In this example, we have created several paragraphs and headings to organize our content into sections and subsections.

Creating Lists

HTML provides two types of lists: ordered lists (<ol>) and unordered lists (<ul>). Ordered lists are used for lists with a specific order, while unordered lists are used for lists without a specific order.

HTML:

<!DOCTYPE html>
<html>
<head>
<title>My First Web Page</title>

</head>
<body>
<h1>Welcome to My First Web Page!</h1>
<p>This is the first paragraph of my web page.</p>
<h2>Section One</h2>
<ol>
<li>Item One</li>
<li>Item Two</li>
<li>Item Three</li>
</ol>
<h2>Section Two</h2>
<ul>
<li>Item One</li>
<li>Item Two</li>
<li>Item Three</li>
</ul>
</body>
</html>

In this example, we have created both an ordered list and an unordered list to display our content in a list format.

Adding Images to Your HTML Page

Images are an important part of web design, and you can easily add them to your HTML page using the 'img' element. The 'img' element is a self-closing tag, which means it doesn't have a closing tag.

Here's an example of how to use the 'img' element to add an image to your page:

HTML:

<img src="https://monkegamer.com/logo.png" alt="MonkeGamer Logo">

In this example, we've used the src attribute to specify the URL of the image we want to display. We've also included an alt attribute, which provides alternative text for the image in case it can't be displayed.

The 'src' attribute is required for the 'img' element, and it should point to the location of your image file on the web. You can use either an absolute URL (like the one we used in the example) or a relative URL (which we'll cover in a future tutorial).

Here are some additional attributes you can use with the 'img' element:

  • 'width': specifies the width of the image in pixels or as a percentage of the parent element's width.
  • 'height': specifies the height of the image in pixels or as a percentage of the parent element's height.
  • 'border': specifies the width of the border around the image.
  • 'align': specifies the alignment of the image with respect to surrounding elements.
  • 'title': specifies additional information about the image (displayed as a tooltip when the mouse hovers over the image).

    • You can include multiple img elements on a single page to display multiple images. Just be sure to specify a unique src attribute for each image.
