Html Basics

April 25, 2023
Written by Daniel Ogden

Introduction

HTML stands for Hypertext Markup Language and is used to create web pages. HTML allows you to define the structure of a web page using a series of tags and attributes. These tags tell the browser how to display the content on the web page.

The Basic Structure of an HTML Document

An HTML document is made up of several parts, including the '<!DOCTYPE>' declaration, the '<html>' element, the '<head> section, the '<body>' section, etc

The '<!DOCTYPE>' declaration is the first line of an HTML document and tells the browser which version of HTML the document is written in.

The '<html>' element is the top-level element in an HTML document and contains all other elements on the page.

The '<head>' section contains metadata about the web page, such as the title of the page, which is displayed in the browser's title bar.

The '<body>' section contains the actual content of the web page, such as text, images, and videos.

HTML Tags and Attributes

HTML tags are used to define different parts of a web page, such as headings, paragraphs, images, and links. HTML tags are enclosed in angle brackets, like this:

String

The 'string' data type is used to represent text values, enclosed in quotes (either single or double). For example:

HTML:

<tagname>content</tagname>

Some HTML tags can also have attributes, which provide additional information about the tag. Attributes are added to the opening tag of an HTML element, like this:

HTML:

<tagname> attribute="value">content</tagname>

Creating Your First Web Page

To create your first web page, you can use a simple text editor like Notepad or TextEdit to write your HTML code. Here is an example of a simple web page:

HTML:

<!DOCTYPE html>
<html>
<head>
<title>My First Web Page</title>
</head>
<body>
<h1>Welcome to My First Web Page!</h1>
<p>This is the first paragraph of my web page.</p>
<p>This is the second paragraph of my web page.</p>
</body>
</html>

Save the file with a '.html' extension and open it in your web browser to see your first web page!

Conclusion

In this tutorial, we have covered the basics of HTML and how it is used to create web pages. We have also looked at the basic structure of an HTML document, HTML tags and attributes, and how to create a simple web page.
